Production Cost Calculator

Estimate creative production costs for a program. Add assets from the rate card, configure management fees, and add localisation costs to see the total production budget.

How to use this calculator
  1. Set the GBP to USD exchange rate if the default (1.26) needs updating. All rate card prices are stored in GBP and converted automatically.
  2. In the Creative Assets table, set the quantity for each asset type you need. Costs are pre-populated from the agency rate card but can be edited. Use Add Custom Asset for items not on the rate card.
  3. The Agency Project Management fee is auto-calculated based on the number of videos, scripts, and banners using the agency's tiered pricing. Toggle manual override to enter a custom amount. Set Additional Revision Rounds if you expect to exceed the 3 included rounds (+15% surcharge per round).
  4. Set Internal Management Fees by entering each role's rate and their allocation as a percentage of creative cost.
  5. Add Localisation countries with the cost as a percentage of the total creative asset cost.
  6. Review the Cost Summary at the bottom for the full production budget breakdown.
